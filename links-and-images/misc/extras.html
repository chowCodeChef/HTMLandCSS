Character Sets

You can use UTF-8 to count in Turkish:

  1. bir
  2. iki
  3. üç
  4. dört
  5. beş

HTML Entities

There are 3 reserved characters in HTML: < > &. You should always use HTML entities to use these three characters.

With UTF-8 anyhow we can use any special character directly in HTML - < > & [ ] { } ' ' " ".
